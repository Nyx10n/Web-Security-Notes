# Cross-Site Request Forgery (CSRF)
<div align=justify>
Cross-Site Request Forgery (CSRF) is a web application vulnerability that tricks an authenticated user into performing unintended actions on a web application without their knowledge or consent.
<br><br>
Unlike attacks that steal credentials or execute malicious JavaScript, CSRF abuses the trust a web application places in an already authenticated user. If the victim is logged in, the browser automatically includes session cookies with the forged request, making it appear as though the request was intentionally made by the legitimate user.
</div>
